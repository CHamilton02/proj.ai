Portfolio Website

Description

Build a simple portfolio website showcasing three sections: a landing page, a "Projects" section with cards displaying brief details, and a "Contact" section with a contact form. Use React for structure, Sass for styling, and ensure it is fully responsive.

Tips

Use React components for each section (e.g., ‹LandingPage />, ‹Projects />, ‹Contact/>).
Leverage Sass features like variables and mixins to create a cohesive design.
Focus on responsive design using media queries in Sass for different screen sizes.
Use Flexbox or CSS Grid for layout.
Deploy your website using GitHub Pages or Vercel.
